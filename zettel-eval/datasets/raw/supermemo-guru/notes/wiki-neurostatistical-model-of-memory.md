## Outline

**Neurostatistical Model of Memory** describes processes underlying the formation of long-term memory. In the model, memories are described by four core properties: [stability](https://supermemo.guru/wiki/Stability "Stability"), [coherence](https://supermemo.guru/wiki/Coherence "Coherence"), [retrievability](https://supermemo.guru/wiki/Retrievability "Retrievability"), and [complexity](https://supermemo.guru/wiki/Complexity "Complexity").

## Origins

The neurostatistical model of memory was based on statistical properties of long-term memory derived from the memory effects of [varied spaced review schedules](https://supermemo.guru/wiki/Algorithm_SM-17 "Algorithm SM-17"). It expands the [two component model of memory](https://supermemo.guru/wiki/Two_component_model_of_memory "Two component model of memory") beyond the molecular interpretation by involving structural and neural aspects of long term changes in properties of memory. The model provides basic functional ingredients needed for the brain to function as a [concept network](https://supermemo.guru/wiki/Concept_network "Concept network").

Neurostatistical Model of Memory resolves the plasticity-stability dilemma

## New concepts

In addition to the skeletal ideas of the [two component model of memory](https://supermemo.guru/wiki/Two_component_model_of_memory "Two component model of memory"), the new contributions of the model are:

- statistical model of [Retrievability](https://supermemo.guru/wiki/Retrievability "Retrievability") based on neural network [interference](https://supermemo.guru/wiki/Interference "Interference")
- model of [Stability](https://supermemo.guru/wiki/Stability "Stability") based on the [translocation of AMPA receptors](https://supermemo.guru/wiki/AMPA_receptors_stabilize_the_dendritic_branch "AMPA receptors stabilize the dendritic branch"), the growth of dendritic spines and synaptic changes in [sleep](https://supermemo.guru/wiki/Sleep "Sleep") (see: [Two-component model of memory stability](https://supermemo.guru/wiki/Two-component_model_of_memory_stability "Two-component model of memory stability"))
- hypothetical model of the [spacing effect](https://supermemo.guru/wiki/Spacing_effect "Spacing effect") based on the clearance of dendritic filopodia (see: [Structural and molecular mechanisms of the spacing effect](https://supermemo.guru/wiki/Structural_and_molecular_mechanisms_of_the_spacing_effect "Structural and molecular mechanisms of the spacing effect"))
- memory [Coherence](https://supermemo.guru/wiki/Coherence "Coherence") model (incl. [memory consolidation in sleep](https://supermemo.guru/wiki/Memory_optimization_in_sleep "Memory optimization in sleep"))
- memory [Complexity](https://supermemo.guru/wiki/Complexity "Complexity") model

## Learning

According to the model, learning would result in dendritic spine tagging, insertion of [AMPA receptors](https://supermemo.guru/wiki/AMPA_receptors_stabilize_the_dendritic_branch "AMPA receptors stabilize the dendritic branch"), and elimination of untagged spines. Inverse tagging might be use to eliminate [interfering](https://supermemo.guru/wiki/Interference "Interference") connections. Learning would then result in a net loss of dendritic spines. In addition, [spacing effect](https://supermemo.guru/wiki/Spacing_effect "Spacing effect") could arise from the need to clear dendritic filopodia from the site undergoing [stabilization](https://supermemo.guru/wiki/Stabilization "Stabilization"). The exploratory growth of filopodia would be a response to reduced postsynaptic activity.
