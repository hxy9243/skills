> de路fud路dle /di藞f蕦dl/ *transitive verb*
> 
> to remove unnecessary elements from a web page, and make it easily readable.

Inspired by my work on [[obsidian-web-clipper|Obsidian Web Clipper]] I created [Defuddle](https://defuddle.md/), an open source library that extracts the main content from any HTML or web page, and converts it to Markdown.

The simplest way to use it is to add `defuddle.md` before any URL, which will return Markdown.
