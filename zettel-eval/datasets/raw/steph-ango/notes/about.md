## Colophon

This site is written and edited in [[obsidian|Obsidian]]. I adhere to the philosophy of [[file-over-app|File over app]]. My plain text Obsidian files are automatically compiled into web pages using [Jekyll](https://jekyllrb.com/) and hosted with [Netlify](https://www.netlify.com/).

The color palette is [[flexoki|Flexoki]], which I created for this site. My Jekyll template is not public, but you can get similar results from [this template](https://github.com/maximevaillancourt/digital-garden-jekyll-template) by Maxime Vaillancourt.

Learn more about [[vault|how I use Obsidian]].
